Online Futsal Booking System
A comprehensive futsal court booking management system with user authentication, real-time availability tracking, payment verification, and admin analytics.

***User Features***

1.User Registration & Login - Secure authentication via phone number

2.Live Availability View - Real-time slot availability with color-coded indicators (Green = Available, Red = Booked)

Intelligent Booking - Automatic conflict prevention with time slot management

4.Payment Integration - QR code payment system with receipt upload

5.Booking History - View all past and upcoming bookings in user profile

6.Cancel Booking - Users can cancel their own reservations

7.Booking Status Check - Public interface to check booking status by name

***Admin Features***

1.Admin Dashboard - Overview of total bookings, pending requests, and revenue statistics

2.Booking Management - Verify payments, confirm bookings, mark as completed, or delete

3.User Management - View all registered users and their activity logs

4.Court Management - Add, modify, or remove futsal courts with dynamic pricing

5.Revenue Reports - Generate monthly/yearly revenue reports with Excel/PDF export

6.Payment Verification - Review and approve payment screenshots

7.User Activity Tracking - Monitor individual user booking patterns
